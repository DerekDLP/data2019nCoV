This vignette briefly demonstrates the use of data from the R package data2019nCoV, available on GitHub. Be sure to reinstall the package to use the latest data, via devtools::install_github("eebrown/data2019nCoV").

This data was manually extracted from the WHO situation reports and stored in raw-data/WHO_SR.csv, which is loaded as the data frame WHO_SR in this package.

Plot example

plot(WHO_SR$Date, WHO_SR$Global.confirmed, 
     main = "Global 2019-CoV Confirmed Cases",
     xlab = "Date",
     ylab = "Confirmed Cases")


gather(WHO_SR, key, value, Japan, RepublicofKorea, VietNam, Singapore, Australia, Malaysia, Cambodia, Philippines, Thailand, Nepal, SriLanka, India, UnitedStatesofAmerica, Canada, Finland, France, Germany, Italy, UnitedArabEmirates) %>%
  ggplot(aes(x=Date, y=value, colour=key)) +
  geom_line() +
  theme(legend.position="bottom") +
  labs(title = "Confirmed Cases Outside of China",
       x = "Date", 
       y = "Confirmed Cases") +
  theme(legend.title = element_blank())
